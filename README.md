### Проверка корректности формата файла резюме
Убедитесь, что у вас установлен Python:
```shell
python --version
```
Установите необходимые зависимости:
```shell
pip install pyyaml
```
Запустите скрипт проверки:
```shell
python scripts/validate_yaml.py
```
